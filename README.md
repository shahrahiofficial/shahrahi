# Shahrahi Educational Repository

This repository is designed to organize educational content, mock exams, study materials, Claude prompts, and scripts for O Level and A Level teaching.

## Directory Structure

- **educational_content/**: Contains various educational resources.
- **mock_exams/**: Contains mock exams for practice.
- **study_materials/**: Contains study materials, notes, and guides.
- **claude_prompts/**: Contains prompts to interact with Claude for educational purposes.
- **scripts/**: Contains scripts for automating tasks or providing additional resources.

## Getting Started

### For O Level and A Level Teaching
1. Clone the repository:
   ```bash
   git clone https://github.com/shahrahiofficial/shahrahi.git
   ```
2. Navigate to the directory:
   ```bash
   cd shahrahi
   ```
3. Explore the folders to find the required resources for your teaching needs.

We hope this repository serves as a valuable resource for educators and students alike!
